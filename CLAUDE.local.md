# Local Overrides

## Environment

- OS: Windows 11
- Shell: PowerShell (not bash — use `;` not `&&` for chaining, `2>$null` not `2>/dev/null`)
- Editor: Antigravity / VS Code
- Node: v20.x

## Dev Preferences

- I prefer `npm` over yarn or pnpm
- Always open localhost:5173 after `npm run dev` — don't just tell me to open it
- When creating new components, use named exports, not default exports

## Personal Rules

- I'm an intermediate coder — give brief explanations alongside code, don't just dump it
- When something is going to take more than 3 steps, ask me to confirm the plan first
- Don't over-engineer. If the simple solution works, use it.
